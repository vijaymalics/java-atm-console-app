# 🏧 ATM Simulation System (Java)

## 📌 Project Overview
ATM Simulation is a Java console-based application that simulates basic ATM operations such as:
- Check Balance
- Deposit Money
- Withdraw Money
- Exit

This project is menu-driven and demonstrates real-world banking logic implementation.

---

## 🚀 Features
✔ Menu-driven program  
✔ Balance checking  
✔ Money deposit functionality  
✔ Money withdrawal with validation  
✔ Loop-based continuous operation  

---

## 🛠 Technologies Used
- Java
- OOPS Concepts
- Scanner Class
- Loops
- Conditional Statements

---

## 📂 How to Run

1. Clone the repository:
   git clone https://github.com/your-username/ATM_Simulation.git

2. Compile:
   javac ATM_Simulation.java

3. Run:
   java ATM_Simulation
